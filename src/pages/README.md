# Page Components

This folder contains page-level (full-screen) React components that represent different routes/views in the application.

## Structure

### Available Pages
- **Dashboard/** - Main dashboard page
- **BeneficiaryManagement/** - Beneficiary management page
- **Reports/** - Reports and analytics page
- **Settings/** - Application settings page

## Page Structure

Each page should follow this structure:

```
PageName/
├── index.jsx      # Page component
├── PageName.css   # Page styles
└── README.md      # Page documentation
```

## Usage

```javascript
import Dashboard from './Dashboard';
```
